MOVIE-IMDB-SCORE
===================================
影片太多不知道要看哪部
在本機目錄下搜尋影片名稱並且取得IMDB分數

### Prerequisites
xampp

### Usage
將目錄移至htdocs/
打上網址 http://localhost/movie/ 就OK了
imdb.php的正規可以自己修正
index.php的遞迴深度也可以自己改
